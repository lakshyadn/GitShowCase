Register a User

Register a GitHub username to generate a portfolio.


POST https://2at5n0ofhc.execute-api.us-east-1.amazonaws.com/dev/register
Content-Type: application/json
Accept: application/json

{
  "username": "your-github-username"
}

Fetch Portfolio Data

Retrieve portfolio data for embedding or display.


POST https://2at5n0ofhc.execute-api.us-east-1.amazonaws.com/dev/showcase
Content-Type: application/json
Accept: application/json

{
  "username": "your-github-username"
}
